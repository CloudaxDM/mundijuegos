## Initializing a New Reflex Project

When starting a new Reflex project or setting up a development environment, you **must** follow the [setup-python-env](skills/setup-python-env/SKILL.md) skill before doing anything else.

Do not skip any steps. Do not assume that a virtual environment or Reflex is already available. Always verify it first by following the skill instructions in order.

Once the environment is ready and Reflex is installed, run:

```bash
reflex init
```

Then continue with the user's request.
